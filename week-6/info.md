# Week 6

## Aim
Implement one storage allocation strategy. This folder uses heap allocation because it is the smallest to remember.

## Files
- `program.c`

## Performing Steps
1. Allocate memory at runtime using `malloc`.
2. Store values in the allocated block.
3. Print the values.
4. Release the memory using `free`.

## How To Run
```powershell
cd week-6
gcc program.c -o program
.\program.exe
```

## Sample Output
```text
Heap values: 10 20 30 40 50
Memory freed
```
